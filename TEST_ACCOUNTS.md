# ProTouchDesign - 테스트 계정 안내

## 📋 개요
ProTouchDesign 시스템의 기능을 체험할 수 있는 테스트 계정입니다.

---

## 👤 테스트 계정

### 통합 테스트 계정
- **이메일**: `test@protouch.com`
- **비밀번호**: `test1234`
- **역할**: 고객 (Customer)
- **접근 가능 페이지**:
  - 고객 대시보드: `/customer/dashboard`
  - 분석 리포트: `/customer/report/1`

**주요 기능**:
- 새로운 웹사이트 분석 요청
- 분석 내역 확인
- AI 분석 리포트 조회
- Figma AI 프롬프트 복사

---

## 🚀 빠른 시작 가이드

### 방법 1: 로그인 페이지 사용
1. `/login` 페이지로 이동
2. "테스트 계정으로 빠른 로그인" 버튼 클릭 (또는 위 정보 직접 입력)

### 방법 2: 데모 링크 사용
1. 랜딩 페이지의 "데모 체험" 섹션으로 이동
2. 원하는 페이지 카드 클릭 (자동 로그인)

---

## ⚠️ 참고사항

- 현재는 프론트엔드만 구현된 상태입니다
- 백엔드 연결은 추후 진행 예정입니다
- 모든 데이터는 Mock 데이터입니다
- 실제 API 호출 없이 작동합니다
