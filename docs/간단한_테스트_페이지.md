# 간단한 테스트 페이지

화면이 안 나올 때 테스트용으로 간단한 컴포넌트를 만들어보세요.

## 테스트 방법

1. `App.tsx`를 간단하게 수정하여 테스트
2. 브라우저에서 확인
3. 점진적으로 복잡한 컴포넌트 추가

## 빠른 해결 방법

AuthContext에서 loading이 너무 오래 걸리면 타임아웃을 설정했습니다.
또한 Supabase 연결 실패 시에도 화면이 표시되도록 했습니다.

### 수정 사항

1. ✅ AuthContext에서 타임아웃 설정 (1초)
2. ✅ Supabase 연결 실패 시에도 loading을 false로 설정
3. ✅ Home 컴포넌트에서 loading 상태 처리
